# RBJ 日本語入力補助 (RecipeBrowserJPChatSearch)

tModLoader mod that helps Japanese / IME input in Recipe Browser (and related search bars).

- **Author:** runohito
- **Version:** 2.1.1
- **Dependency:** [Recipe Browser](https://github.com/JavidPack/RecipeBrowser) (required)
- **Side:** Client

Steam Workshop publication: use the Workshop page / local Mod Sources build.
This repository holds the v2.1.1 source snapshot.

## Build

Open the folder under tModLoader `ModSources`, or build with tModLoader's rebuild tools with Recipe Browser available as a mod reference.
